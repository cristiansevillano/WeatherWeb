# WeatherWeb
Web simple para ver el tiempo basada en la API de OpenWeatherMap.

Caracteristicas:

    · API
    · Proxy
    · Event Listeners
    · Cambio de Celsius/Fahrenheit
    · Iconos animados de Skycons
    · Flex-box
